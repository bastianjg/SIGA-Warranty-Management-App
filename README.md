###### \# SIGA - Smart Warranty Management System



SIGA is a mobile application developed with Flutter and Spring Boot that allows users to manage product warranties through automatic receipt digitization using OCR technology.



\## Features



\- User registration and authentication

\- Email verification

\- JWT-based security

\- OCR receipt scanning

\- Warranty management

\- Local notifications

\- User profile management



\## Tech Stack



\### Frontend

\- Flutter



\### Backend

\- Spring Boot

\- Spring Security

\- JWT



\### Database

\- MySQL
